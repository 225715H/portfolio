TODO アプリ

アプリについて( GitHub )

このアプリは、ハンズオンで作成したTODOアプリです。シンプルなデザインと直感的な操作で、タスクの追加、削除、完了を行うことができます。

背景

大学では、pythonを使う機会が多かったため、使い慣れているpytonでアプリを作成したいと思い、Djangoを使ってTODOアプリを作成しました。

使用技術

python Django html bootstrap

苦労した点・工夫した点

苦労した点は、実際にデプロイするときのエラーの対処です。 デプロイすること自体初めてということもあり、エラーが出た際に、どのように対処すればいいのかわからず、ChatGPTを使って、いろんな方法を試して解決しました。

ハンズオン形式で作成したため、特別な工夫は多くありませんが、あえて挙げるなら、デザインをシンプルにすることで直感的な操作性を重視しました。

学んだこと

自主的に初めて取り組んだアプリ開発であったため、アプリ開発の大まかな流れを学ぶことができました。 また、バージョン管理や全て自分で進めていくという点で、自分の力でアプリを作成することの難しさを実感しました。
