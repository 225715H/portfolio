My Word App

アプリについて ( GitHub )

このアプリは、大学の授業で制作した単語帳アプリです。 気になったり、わからない単語を調べながら実際に追加していくことで、自身に合わせた単語帳を作成できます。

背景

チームで、大学生になって初めてコンピュータサイエンスなどを学び、難しい用語が多いと話し合った際に、 用語を覚えるための単語帳のようなアプリがあれば便利だというアイデアを得て、チームで開発しました。

使用技術

python tkinter

苦労した点・工夫した点

苦労した点は、メンバー全員が初めてのチーム開発だったため、コミュニケーションの取り方に苦労したことです。 そのため、4人のチームを2人ずつのペアに分け、ペアプログラミングを行いながら、アジャイル手法を取り入れて進めました。

工夫した点は、ジャンルが増えたり、単語が増えたりしても、使いやすいように、シンプルなデザインにすることで、直感的に操作できるようにしました。 また、理解度をつけることで、学習効率を高めることができるようにしました。

学んだこと

初めてのチーム開発であったため、コミュニケーションをとることがいかに大事かを学びました。 また、使ったライブラリ(tkinter)も初めて使うものであったため、調べながら進めていくことの面白さを知ることができました。
