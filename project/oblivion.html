OBLIVION

アプリについて( Apple Store )

このアプリは、翻訳と単語帳を組み合わせた学習アプリで、独自のアルゴリズムを用いて、ユーザの理解度に合わせた問題を出題します。 従来の単語帳とは違って、自分自身で単語帳を作成するだけではなく、翻訳した単語をそのまま単語帳に登録することができ、効率よく学習を進めることができます。

背景

友人が英語の勉強をしている際に、単語を翻訳してもすぐ忘れるという課題から翻訳した単語をそのまま単語帳に登録できれば便利だというアイデアを得て、その友人と共同で開発しました。

使用技術

TypeScript React Native Expo Discord GitHub Figma

苦労した点・工夫した点

苦労した点は、アプリの仕様を決めていくことでした。 最初は、翻訳した単語をそのまま単語帳に登録することができるというアイデアだけで、そこから色んな機能の案を出し合いましたが、 実装のハードルが高かったり、当初の目的から外れてしまうものが多かったため、何度も議論を重ねて最終的な仕様を決定しました

工夫した点は、ユーザが使いやすいように、シンプルなデザインにすることで、直感的に操作できるようにしました。 また、 コードの可読性を高めるために、atomic designを取り入れ、コンポーネントを細かく分割することで、コードの可読性を高めました。

学んだこと

アプリ開発に関して、完璧を作るよりも完成させることが大切だということを身をもって感じました。 そのため、仕様に関しては、最低限の機能を実装し、周りのフィードバックや実際に使ってみて、必要な機能を追加していくことが重要だと学びました。 また、共同開発したため、コミュニケーションをとりながら、仕様の決定や実装を進めることの大切さを学びました。
